# Security Policy

## Supported Scope

TradeSpine is currently in planning/architecture phase. Security reports are
still welcome for:

- repository integrity and supply-chain risks
- unsafe execution patterns in MQL5 source and templates
- documentation or process gaps that can cause unsafe trading behavior

## Reporting A Vulnerability

Please do not open public issues for sensitive vulnerabilities.

Report privately using one of the following:

- GitHub Security Advisories (preferred, once enabled on remote)
- direct maintainer contact

Include:

- affected files and version/commit
- reproduction steps
- impact and severity estimate
- suggested mitigations (if available)

Acknowledgment target: within 5 business days.
