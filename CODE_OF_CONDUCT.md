# Code Of Conduct

This project is committed to a respectful, harassment-free, and inclusive
environment for all contributors.

## Our Standards

- Be respectful and constructive in discussions and reviews.
- Assume good intent and focus on technical outcomes.
- Avoid personal attacks, discrimination, and harassment.
- Keep feedback specific, actionable, and professional.

## Unacceptable Behavior

- Harassment, abuse, or hateful language.
- Trolling, intimidation, or personal insults.
- Sharing private information without permission.
- Deliberate disruption of discussions or workflows.

## Scope

This code of conduct applies in project spaces, including issues, pull requests,
discussions, and direct collaboration related to this repository.

## Reporting

If you experience or witness unacceptable behavior, report it to the
maintainers via the contact channels documented in [SECURITY.md](SECURITY.md).

Project maintainers may remove, edit, or reject comments, commits, code, wiki
edits, issues, and other contributions not aligned with this code of conduct.
