# Test-Driven Development Guides

Layer 7 maps approved SPEC components to executable test cases and thresholds.

## Current Scope

- Code-deliverable TDD guides exist for SPEC-01 through SPEC-07 and SPEC-09 through SPEC-11.
- TDD-08 is intentionally absent because SPEC-08 is documentation/process governance scope.
- Each TDD guide links to its parent SPEC and downstream IPLAN ownership.

## Entry Points

- [TDD-00_index.md](TDD-00_index.md) is the canonical Layer 7 index.
- Each `TDD-NN_*` folder contains the YAML guide, readable rendering, and audit report.
